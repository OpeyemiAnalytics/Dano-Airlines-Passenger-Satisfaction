# ✈️ Flight Risk: Dano Airlines Passenger Satisfaction Analysis

## Project Overview
This project analyzes passenger satisfaction data for Dano Airlines, 
a UK-based airline headquartered in London. The latest survey results 
revealed the satisfaction rate dropped below 50% for the first time ever. 

As the Data Analyst, my task was to identify key drivers of dissatisfaction 
and recommend a data-driven strategy to improve satisfaction rates.

## Dataset
- **Source:** Digitaleydrive Data Analytics Bootcamp
- **Size:** 129,880 passenger records
- **Columns:** 24 fields including passenger demographics, 
flight details, service ratings and satisfaction status

## Tools Used
- **Power BI Desktop** - Data cleaning, modeling & visualization
- **Power Query** - Data transformation
- **DAX** - Calculated measures

## Data Cleaning Steps
- Verified 129,880 rows with zero duplicate records
- Replaced 393 null values in Arrival Delay with 0 
(event-driven field - no record means no delay occurred)
- Created Age Group column (5 categories)
- Created Distance Group column (3 categories)
- Confirmed all data types were correct

## Key Findings
- Overall satisfaction rate: **43.4%** - below 50% crisis threshold
- Economy class has **47K dissatisfied** vs only 11K satisfied
- Personal travelers almost entirely dissatisfied - 36K vs 4K
- First-time customers leaving unhappy - 18K dissatisfied vs 6K
- WiFi Service scored lowest at **2.7/5** across all services
- Senior Professionals are the ONLY age group where satisfaction 
outweighs dissatisfaction

## Recommendations
1. Prioritize Economy and Economy Plus cabin experience urgently
2. Invest urgently in WiFi infrastructure - lowest rated service at 2.7/5
3. Design targeted retention campaigns for First-time travelers
4. Create personalised experience packages for Personal travelers
5. Address Young Adult dissatisfaction through entertainment and comfort upgrades
